Introduction To Css

Css(Cascading Sytle Sheet): is a styling language used to add style/ design to html code when creating a Web Pages. it works in conjuction with Html to define how elements like text, images and other contents are displayed on the pages

Ways To Write CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS: involves add CSS directly within an Html element using the style attribute. it is use when you need to apply a Unique style to a single element

Internal CSS: is defined within a style tag inside the head section of the Html element. This method is use when you want to style a single Html document.

External CSS: is the best way to write CSS. It is define in a seperate CSS file and linked to a Html document using the link tag. This method is ideal for styling multiple pages consistently.
